CSS Types for Beginners

1. External CSS

External CSS is stored in a separate file and linked to HTML.

This box is styled with external CSS from CSS_Video_1.css

2. Internal CSS

Internal CSS is written inside the HTML file in the head section.

This box is styled with internal CSS

3. Inline CSS

Inline CSS is applied directly to an HTML element.

This box is styled with inline CSS
